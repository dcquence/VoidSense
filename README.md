# VoidSense
Built off the default dark theme. Edited colors in the CSS files and directly in certain widjets. Changed the logo back to the older version.

# Installation
Access the OPNsense device via ssh or something like WinSCP, and add the VoidSense folder into /usr/local/opnsense/www/themes. Overwrite widget files in /usr/local/opnsense/www/js/widgets
You will now have a new theme option in Settings/General
